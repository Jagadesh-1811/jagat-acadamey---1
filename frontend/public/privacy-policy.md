## 1. Introduction

## 2. Information We Collect

### 2.1 Personal Information You Provide

When you register and use our platform, we collect:

| Information Type | Purpose |
|-----------------|---------|
| **Name** | Account identification and personalization |
| **Email Address** | Account login, notifications, and communications |
| **Password** | Account security (stored encrypted) |
| **Profile Photo** | Personalization and user identification |
| **Phone Number** (optional) | Contact and support purposes |

We collect learning-related information including:

- **Course Enrollments:** Courses you enroll in

- **Quiz Responses:** Answers and scores on assessments
- **Grades:** Academic performance data
- **Certificates Earned:** Completion certificates issued
- **Resume Data:** Information added to your resume builder

### 2.3 Communication Data

- **Voice Room Recordings:** Audio session metadata (not content)
- **Chat History:** AI chatbot conversations (stored locally and/or server)
- **Feedback & Reviews:** Course reviews and platform feedback
- **Support Requests:** Issues and queries you submit

### 2.4 Automatically Collected Information

| Data Type | Description |
|-----------|-------------|
| **IP Address** | For security and analytics |
| **Device Information** | Browser type, operating system |
| **Access Times** | Login and session timestamps |
| **Page Views** | Platform usage patterns |
| **Cookies** | Session management and preferences |

## 5. Data Security

### 5.1 Security Measures

We implement industry-standard security measures:

- **Encryption:** Passwords are hashed using bcrypt
- **HTTPS:** All data transmitted is encrypted via SSL/TLS
- **JWT Tokens:** Secure authentication tokens
- **Access Controls:** Role-based access restrictions
- **Regular Updates:** Security patches and updates

### 5.2 Your Responsibilities

- Keep your password confidential
- Use strong, unique passwords
- Log out from shared devices
- Report suspicious account activity immediately

### 5.3 Data Breach Response

In case of a data breach:
- We will notify affected users within 72 hours
- We will take immediate steps to contain the breach
- We will conduct a thorough investigation
- We will implement measures to prevent future breaches
